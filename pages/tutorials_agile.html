    About the tutorial

    Agile methodology has became the most popular system development methodology for Web and mobile application development and is one of the most in-demand knowledge and skills for IS students. The goal of this tutorial is to showcase how to teach agile methodology, in particular, SCRUM, using a combination of lectures, simulation games, and system demonstration and lab. The related materials have been taught many times at both undergraduate and graduate levels with collaboration from industry partner such as JPMorgan Chase and well received by both students and faculty.

     

    The key planned activities for this tutorial include:

    • Introduction lecture on Agile and SCRUM
    • SCRUM simulation game
    • Github system introduction and demo
    • Related research projects using Github data

     

    The target audience is faculty and researchers who are interested in teaching agile methodology to both undergraduate and graduate students in a highly interactive and entertaining manner. In addition, this tutorial will also benefit scholars who are interested in learning about Github system for managing SCRUM projects and using Github data for analytical research projects.
